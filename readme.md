> [Kap](https://github.com/wulkano/kap) plugin - Share GIFs on [gfycat](https://gfycat.com)


## Install

In the `Kap` menu, go to `Preferencesâ€¦`, select the `Plugins` pane, find this plugin, and toggle it.
